用Markdown优雅的写文章

写在前面

记录第一次真正的使用Markdown来编辑一篇博客，练练手，希望通过这篇博客能够熟练掌握。感兴趣的小伙伴可以跟着我一起呀o(￣▽￣)ブ。那就废话不多说，我们开始吧。

简介

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。 Markdown具有一系列衍生版本，用于扩展Markdown的功能，如表格、脚注、内嵌HTML等等……●-●

Markdown 语法初见

分级标题

直接输入1次#，并按下space后，将生成1级标题。
输入2次#，并按下space后，将生成2级标题。
以此类推，总共6级标题，有助于使用TOC语法后生成一个完美的目录。

斜体和粗体

使用*和**表示斜体和粗体。 我是斜体，我是粗体

外链接

使用 [描述](链接地址) 为文字增加外链接。
示例：
这是我的博客主页的链接。

生成一个适合你的列表

无序列表

网页将会表现为这种格式：

  • Red
  • Green
  • Blue

有序列表

使用数字和点表示有序列表。注意数字和点后面要有空格。 示例：

  1. 光辉岁月
  2. 不再犹豫
  3. 真的爱你

添加注脚

安装插件：

用法如下：
一个具有注脚的文本。[^1]
[^1]注脚的解释
示例：
一个具有注脚的文本。[1]

创建一个表格

一个简单的表格是这么创建的：

项目 Value
电脑 $1600
手机 $12
导管 $1

文字引用

引用块

使用>表示文字引用。
示例：

“在你为人类做出贡献之前，你应当耻于死去” - - 伟大的美国教育家霍瑞斯·曼
